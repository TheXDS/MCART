License

Obtiene el texto de la licencia de este Plugin.

Declaration
public virtual License License { get; }
Property Value
Type Description
License

El valor del atributo LicenseFileAttribute, EmbeddedLicenseAttribute o LicenseTextAttribute, cualesquiera esté definido primero para este Plugin, o en caso de no establecer ninguno de los atributos, se devolverá un texto de licencia no establecida, o un mensaje de error junto con el StackTrace en caso de no poder obtener la información de licencia debido a una excepción.

note

La licencia será buscada en el mismo orden establecido en el apartado de valor, y se devolverá únicamente un atributo al ser encontrado.

TargetMcartVersion

Determina la versión objetivo de MCART para este Plugin.

Declaration
public virtual Version TargetMcartVersion { get; }
Property Value
Type Description
System.Version

El valor del atributo TargetMCARTVersionAttribute definido como la versión de MCART para la cual este Plugin ha sido diseñado.
Remarks

Si este Plugin se intenta cargar en una versión no soportada de MCART, la carga es abortada.

caution

(NO RECOMENDADO) En caso de que desee cargar un plugin sin verificación de compatibilidad, cree un nuevo PluginLoader utilizando una nueva instancia de RelaxedPluginChecker como verificador de Plugin. Tome en cuenta que, es posible que el programa falle si se intenta utilizar un plugin no compatible, debido a los posibles cambios de API entre versiones de MCART.
