Class TamperException

Se produce cuando una función soportada detecta una de las siguientes situaciones:

  • Valores de retorno alterados inesperadamente.
  • Valor de retorno fuera del rango conocido esperado de una función.
  • Corrupción de memoria no capturada por CLR.
  • Modificación externa de valores internos protegidos de la aplicación.
  • Acceso inesperado a métodos internos.
