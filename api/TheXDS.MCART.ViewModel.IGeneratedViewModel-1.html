Name Description
T

Tipo de interfaz de acceso a implementar.
Remarks

Se recomienda que esta interfaz sea implementada de forma abstracta, ya que la clase ViewModelFactory invalidará estos miembros, por lo que las implementaciones directas de estos miembros en la clase base se ignorarán por medio de Shadowing.
