CenturyEpoch

Obtiene un System.DateTime que representa el inicio del tiempo del siglo 20.

Declaration
public static readonly DateTime CenturyEpoch
Field Value
Type Description
System.DateTime
Remarks

Se define el Century Epoch como el punto de inicio del tiempo para muchos dispositivos de 32 bits que no siguen el estándar de Unix, y se establece como el primero de enero del año 1900. El estándar se definía por medio del tiempo GMT, en su ausencia se utiliza UTC como un substituto suficientemente cercano.
