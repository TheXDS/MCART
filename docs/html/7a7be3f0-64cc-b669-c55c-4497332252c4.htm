RandomExtensionsRndFileName (Método )

[Ésta es documentación preliminar y está sujeta a cambios.]

Genera un nombre de archivo aleatorio que no se encuentre en uso en el directorio actual.

Parámetros

r
Tipo: SystemRandom
Instancia del objeto Random a utilizar.
length (Optional)
Tipo: SystemByte
Longitud del nombre de archivo.

Valor devuelto

Tipo: String
Un nombre de archivo aleatorio que no corresponde a ningún archivo en el directorio actual.

Nota sobre el uso

En Visual Basic y C#, se puede llamar a este método como a un método de instancia en cualquier objeto de tipo Random. Cuando emplee la sintaxis de método de instancia para llamar a este método, omita el primer parámetro. Para obtener más información, vea Métodos de extensión (Visual Basic) o Métodos de extensión (Guía de programación C#).
Excepciones
ExcepciónCondición
ArgumentOutOfRangeException Se produce si length es inferior a 0, o superior a el máximo admitido por el sistema operativo.
