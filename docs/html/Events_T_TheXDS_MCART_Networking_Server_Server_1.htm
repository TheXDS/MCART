ServerTClient (Eventos)

[Ésta es documentación preliminar y está sujeta a cambios.]

El tipo genérico ServerTClient expone a los siguientes miembros.

Eventos
  NombreDescripción
ClientAccepted
Ocurre cuando el protocolo ha aceptado al nuevo cliente.
(Se hereda de Server.)
ClientConnected
Ocurre cuando un nuevo cliente se conecta al servidor.
(Se hereda de Server.)
ClientDisconnected
Ocurre cuando un cliente se desconecta correctamente.
(Se hereda de Server.)
ClientFarewell
Ocurre cuando el protocolo despide a un cliente.
(Se hereda de Server.)
ClientLost
Ocurre cuando un cliente se desconecta inesperadamente.
(Se hereda de Server.)
ClientRejected
Ocurre cuando el protocolo rechaza al nuevo cliente.
(Se hereda de Server.)
ServerStarted
Ocurre cuando el servidor es iniciado.
(Se hereda de Server.)
ServerStopped
Ocurre cuando el servidor es detenido.
(Se hereda de Server.)
