Parámetros

uri
Tipo: SystemUri
Url del archivo. Debe ser una ruta http válida.
stream
Tipo: System.IOStream
Stream en el cual se almacenará el archivo.
reportCallback
Tipo: SystemActionNullableInt64, Int64
Delegado que permite reportar el estado de esta tarea. El primer parámetro devolverá la cantidad de bytes recibidos, o si stream no es capaz de reportar su tamaño actual. El segundo parámetro devolverá la longitud total de la solicitud, o -1 si el servidor no reportó el tamaño de los datos a recibir.
polling
Tipo: SystemInt32
Intervalo en milisegundos para reportar el estado de la descarga.

Valor devuelto

Tipo: Task
Un Task que representa a la tarea en ejecución.
