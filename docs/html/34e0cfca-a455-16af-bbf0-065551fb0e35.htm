IEnumerableExtensions (Métodos)

[Ésta es documentación preliminar y está sujeta a cambios.]

El tipo IEnumerableExtensions expone a los siguientes miembros.

Métodos
  NombreDescripción
CopyT
Ontiene una copia de los elementos de este IEnumerableT
PickT
Selecciona un elemento aleatorio de esta lista.
PopT
Devuelve el último elemento en la lista, quitándolo.
PopFirstT
Devuelve el primer elemento en la lista, quitándolo.
RotateT
Rota los elementos de un arreglo, lista o colección.
ShiftT
Desplaza los elementos de un arreglo, lista o colección.
ShuffleT(IEnumerableT, Int32)
Desordena los elementos de un IEnumerableT.
ShuffleT(IEnumerableT, Int32, Int32, Int32)
Desordena los elementos del intervalo especificado de un IEnumerableT.
ShuffledT(IEnumerableT, Int32)
Devuelve una versión desordenada del IEnumerableT sin alterar la colección original.
ShuffledT(IEnumerableT, Int32, Int32, Int32)
Devuelve una versión desordenada del intervalo especificado de elementos del IEnumerableT sin alterar la colección original.
ToExtendedListT
Convierte una colección a ListT
