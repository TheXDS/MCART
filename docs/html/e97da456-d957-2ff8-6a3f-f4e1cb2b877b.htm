UIRender (Método (FrameworkElement, Size, Size, Int32))

[Ésta es documentación preliminar y está sujeta a cambios.]

Crea un mapa de bits de un FrameworkElement estableciendo el tamaño en el cual se dibujará el control, por lo que no necesita haberse mostrado en la interfaz de usuario.

Parámetros

f
Tipo: System.WindowsFrameworkElement
FrameworkElement a renderizar.
inSize
Tipo: System.WindowsSize
Tamaño del control a renderizar.
outSize
Tipo: System.WindowsSize
Tamaño del canvas en donde se renderizará el control.
dpi
Tipo: SystemInt32
Valor de puntos por pulgada a utilizar para crear el mapa de bits.

Valor devuelto

Tipo: RenderTargetBitmap
Un objeto RenderTargetBitmap que contiene una imagen renderizada de f.

Nota sobre el uso

En Visual Basic y C#, se puede llamar a este método como a un método de instancia en cualquier objeto de tipo FrameworkElement. Cuando emplee la sintaxis de método de instancia para llamar a este método, omita el primer parámetro. Para obtener más información, vea Métodos de extensión (Visual Basic) o Métodos de extensión (Guía de programación C#).
