Plugin (Métodos)

[Ésta es documentación preliminar y está sujeta a cambios.]

El tipo Plugin expone a los siguientes miembros.

Métodos
  NombreDescripción
About
Muestra información del IPlugin.
Equals (Se hereda de Object.)
Finalize
Realiza algunas tareas previas a la destrucción de esta instancia de Plugin por el colector de basura.
(Reemplaza a ObjectFinalize.)
GetHashCode (Se hereda de Object.)
GetType (Se hereda de Object.)
GetUIMenu
Genera un MenuItem a partir de las interacciones del IPlugin.
GetUIPanelT, PanelT
Obtiene la Interfaz de interacción de un IPlugin como un PanelT cuyas acciones son controles de tipo T.
MemberwiseClone (Se hereda de Object.)
MinRTVersion
Determina la versión mínima de MCART necesaria para este Plugin.
RaiseFailed
Genera el evento PluginLoadFailed.
RaiseFinalizing
Genera el evento PluginFinalizing.
RequestUIChange
Genera el evento UIChangeRequested.
TargetRTVersion
Determina la versión objetivo de MCART necesaria para este Plugin.
ToString (Se hereda de Object.)
UIPanelT, PanelT
Convierte el PluginInteractions en un Panel, especificando el tipo de controles a contener para cada InteractionItem.
