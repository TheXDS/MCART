ISliceGraph (Interfaz)

[Ésta es documentación preliminar y está sujeta a cambios.]

Expone una serie de métodos de redibujado disponibles para un control que acepte un Slice.

public interface ISliceGraph
Public Interface ISliceGraph

El tipo ISliceGraph expone a los siguientes miembros.

Propiedades
  NombreDescripción
Slices
Obtiene un listado de los Slice que conforman el set de datos de este ISliceGraph.
SubLevelsShown
Obtiene o establece la cantidad de sub-niveles a mostrar en este ISliceGraph.
Total
Obtiene el total general de los datos de este ISliceGraph.
TotalVisible
Obtiene o establece un valor que determina si se mostrarán los totales de los puntos y el total general de los datos.
