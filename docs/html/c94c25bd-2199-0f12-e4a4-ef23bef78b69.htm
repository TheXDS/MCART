Plugin (Eventos)

[Ésta es documentación preliminar y está sujeta a cambios.]

El tipo Plugin expone a los siguientes miembros.

Eventos
  NombreDescripción
PluginFinalized
Se produce cuando un Plugin ha sido finalizado.
PluginFinalizing
Se produce cuando un Plugin va a ser finalizado.
PluginLoaded
Se produce cuando un Plugin ha sido cargado.
PluginLoadFailed
Se produce cuando un Plugin no pudo ser cargado.
UIChangeRequested
Se produce cuando un Plugin solicita que se actualice su interfaz gráfica, en caso de contenerla.
