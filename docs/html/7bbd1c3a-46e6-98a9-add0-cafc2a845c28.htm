IGraph (Propiedades)

[Ésta es documentación preliminar y está sujeta a cambios.]

El tipo IGraph expone a los siguientes miembros.

Propiedades
  NombreDescripción
Colorizer
Obtiene o establece un IGraphColorizer opcional a utilizar para establecer los colores de las series.
Title
Obtiene o establece el título de este IGraph.
TitleFontSize
Obtiene o establece el tamaño de fuente a aplicar al título.
ToolTipFormat
Obtiene o establece un formato opcional de etiquetado flotante para aplicar a los puntos de datos o series dibujadas en este IGraph.
