Public Shared Function GetTypes(Of T) As IEnumerable(Of Type)
Request Example View Source

Parámetros de tipo

T
Interfaz o clase base a buscar.

Valor devuelto

Tipo: IEnumerableType
Una lista de tipos de las clases que implementan a la interfaz o que heredan a la clase base T dentro de CurrentDomain.
Observaciones
Esta función obtiene todos los tipos (privados y públicos) definidos dentro de todos ls ensamblados dentro del dominio actual. Para obtener únicamente aquellos tipos exportados públicamente, utilice PublicTypes(Type), PublicTypes(Type, AppDomain), PublicTypesT o PublicTypesT(AppDomain).
