MCART.Controls (Espacio de nombres)

[Ésta es documentación preliminar y está sujeta a cambios.]

El espacio de nombres MCART.Controls contiene una librería de controles/widgets disponibles para aplicaciones de GUI.
Clases
  ClaseDescripción
BarStepper
Este control permite reportar visualmente el progreso de una tarea en una barra de estado; opcionalmente, mostrando además un botón para cancelar dicha tarea.
BusyIndicator
Control simple que indica al usuario que la aplicación está ocupada.
Extensions
Extensiones especiales para Slice.
HealthColorizer
Coloreado de salud.
HeatColorizer
Coloreado de temperatura.
LightGraph
Control/Widget de gráficos ligero.
Misc
Funciones misceláneas para controles/widgets.
NavigationBar
Permite la navegación y búsqueda entre una lista de elementos por medio de la GUI.
ProgressRing
Permite representar un valor porcentual en un anillo de progreso.
RandomColorizer
Coloreado aleatorio.
RingGraph
RingGraph
SearchBox
Cuadro de texto simple con marca de agua optimizado para búsquedas.
Series
Representa una serie de valores que pueden ser representados en un control TODO.
Slice
Representa una sección de un ISliceGraph.
Interfaces
  InterfazDescripción
IGraph
Define una serie de métodos y propiedades a implementar por un control que permita mostrar gráficos de cualquier tipo.
IGraphColorizer
Describe una serie de métodos a implementar por clases que permitan colorear una colección de IGraphData.
IGraphData
Determina una serie de propiedades a implementar por clases que puedan ser utilizadas en controles/widgets IGraph.
ISliceGraph
Expone una serie de métodos de redibujado disponibles para un control que acepte un Slice.
