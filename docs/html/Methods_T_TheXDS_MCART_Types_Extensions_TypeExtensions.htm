TypeExtensions (Métodos)

[Ésta es documentación preliminar y está sujeta a cambios.]

El tipo TypeExtensions expone a los siguientes miembros.

Métodos
  NombreDescripción
AreAllAssignable(Type, IEnumerableType)
Comprueba si todos los tipos son asignables a partir del tipo source.
AreAllAssignable(Type, Type)
Comprueba si todos los tipos son asignables a partir del tipo source.
Assignables(Type, IEnumerableType)
Enumera los tipos asignables a partir de source.
Assignables(Type, Type)
Enumera los tipos asignables a partir de source.
Default
Equivalente programático de default, obtiene el valor predeterminado del tipo.
Implements(Type, Type)
Determina si el tipo implementa a baseType.
Implements(Type, Type, Type)
Determina si el tipo implementa a baseType con los argumentos de tipo genérico especificados.
ImplementsT(Type)
Determina si el tipo implementa a T.
IsAnyAssignable(Type, IEnumerableType)
Comprueba si alguno de los tipos especificados es asignable a partir del tipo source.
IsAnyAssignable(Type, Type)
Comprueba si alguno de los tipos especificados es asignable a partir del tipo source.
IsInstantiable(Type)
Obtiene un valor que determina si el tipo es instanciable.
IsInstantiable(Type, IEnumerableType)
Obtiene un valor que determina si el tipo es instanciable.
New(Type)
Inicializa una nueva instancia del tipo en runtime especificado.
New(Type, Object)
Inicializa una nueva instancia de un objeto con un constructor que acepte los argumentos provistos.
NewT(Type)
Inicializa una nueva instancia del tipo dinámico especificado, devolviéndola como un T.
NewT(Type, Object)
Inicializa una nueva instancia de un objeto con un constructor que acepte los argumentos provistos.
TypeName
Obtiene un nombre personalizado para un tipo.
