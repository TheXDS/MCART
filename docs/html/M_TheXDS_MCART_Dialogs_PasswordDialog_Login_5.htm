Parámetros

loginValidator
Tipo: TheXDS.MCART.Security.PasswordLoginValidator
Delegado de validación de inicio de sesión. Debe tratarse de un TaskTResult que devuelve un Boolean, si el inicio de sesión fue exitoso, en caso contrario.
maxTries
Tipo: SystemInt32
Cantidad máxima de intentos. El inicio de sesión se cancela si el usuario excede esta cantidad de intentos inválidos.

Valor devuelto

Tipo: NullableBoolean
si el inicio de sesión fue exitoso, en caso contrario o al exceder la cantidad máxima de intentos, o si el inicio de sesión fue cancelado.
