ISliceGraph (Interfaz)

[Ésta es documentación preliminar y está sujeta a cambios.]

Expone una serie de métodos de redibujado disponibles para un control que acepte un Slice.

Espacio de nombres:  TheXDS.MCART.Controls
Ensamblado:  TheXDS.MCART.WPF (en TheXDS.MCART.WPF.dll) Versión: 0.8.10.0
Sintaxis
C#
VB

public interface ISliceGraph : IGraph
Public Interface ISliceGraph
	Inherits IGraph

El tipo ISliceGraph expone a los siguientes miembros.

Propiedades
  NombreDescripción
Colorizer
Obtiene o establece un IGraphColorizer opcional a utilizar para establecer los colores de las series.
(Se hereda de IGraph.)
Slices
Obtiene un listado de los Slice que conforman el set de datos de este ISliceGraph.
SubLevelsShown
Obtiene o establece la cantidad de sub-niveles a mostrar en este ISliceGraph.
Title
Obtiene o establece el título de este IGraph.
(Se hereda de IGraph.)
TitleFontSize
Obtiene o establece el tamaño de fuente a aplicar al título.
(Se hereda de IGraph.)
ToolTipFormat
Obtiene o establece un formato opcional de etiquetado flotante para aplicar a los puntos de datos o series dibujadas en este IGraph.
(Se hereda de IGraph.)
Total
Obtiene el total general de los datos de este ISliceGraph.
TotalFormat
Obtiene o establece el formato a aplicar a la etiqueta de total del ISliceGraph.
TotalVisible
Obtiene o establece un valor que determina si se mostrarán los totales de los puntos y el total general de los datos.

Métodos
  NombreDescripción
DrawMe

DrawMyChildren
Dibuja únicamente a los hijos del Slice.
Init
Realiza tareas finales de inicialización del control.
(Se hereda de IGraph.)
Redraw
Solicita al control volver a dibujarse en su totalidad.
(Se hereda de IGraph.)
