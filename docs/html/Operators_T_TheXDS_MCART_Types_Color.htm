Color (Operadores y Conversiones de Tipos)

[Ésta es documentación preliminar y está sujeta a cambios.]

El tipo Color expone a los siguientes miembros.

Operadores
  NombreDescripción
Addition
Adds a Color to a Color, yielding a new Color.
Division
Realiza una mezcla entre los colores especificados.
Equality
Determina si dos instancias de Color son iguales.
(Color to Color)
Convierte implícitamente una estructura Color en un Color.
(Color to Color)
Convierte implícitamente una estructura Color en un Color.
(Color to Color)
Convierte implícitamente una estructura Color en un Color.
(Color to Brush)
Convierte implícitamente una estructura Color en un Brush.
(Color to Color)
Convierte implícitamente una estructura Color en un un Color.
Inequality
Determina si dos instancias de Color son distintas.
Multiply
Computes the product of left and right, yielding a new Color.
Subtraction
Sustrae un Color de un Color, dando como resultado un Color.
