CommonCondense (Método )

[Ésta es documentación preliminar y está sujeta a cambios.]

Condensa un arreglo de String en una sola cadena.

Espacio de nombres:  MCART
Ensamblado:  MCART (en MCART.dll) Versión: 0.7.0.0
Sintaxis
C#
VB
Copy
public static string Condense(
	this IEnumerable<string> str,
	string separation = ""
)
<ExtensionAttribute>
Public Shared Function Condense ( 
	str As IEnumerable(Of String),
	Optional separation As String = ""
) As String

Parámetros

str
Tipo: System.Collections.GenericIEnumerableString
Arreglo a condensar.
separation (Optional)
Tipo: SystemString
Establece una cadena de separación entre los elementos de la cadena. Si de omite, se utilizará un espacio en blanco.

Valor devuelto

Tipo: String
Una cadena compuesta por los elementos de la colección, separados por el caracter especificado.

Nota sobre el uso

En Visual Basic y C#, se puede llamar a este método como a un método de instancia en cualquier objeto de tipo IEnumerableString. Cuando emplee la sintaxis de método de instancia para llamar a este método, omita el primer parámetro. Para obtener más información, vea Métodos de extensión (Visual Basic) o Métodos de extensión (Guía de programación C#).
