IPlugin (Interfaz)

[Ésta es documentación preliminar y está sujeta a cambios.]

Define una interfaz básica para crear Plugins administrados por MCART

Espacio de nombres:  MCART.PluginSupport
Ensamblado:  MCART (en MCART.dll) Versión: 0.7.0.0

public interface IPlugin
Public Interface IPlugin

El tipo IPlugin expone a los siguientes miembros.

Propiedades
  NombreDescripción
Author
Devuelve el autor del IPlugin
Copyright
Devuelve el Copyright del IPlugin
Description
Devuelve una descripción del IPlugin
HasInteractions
Devuelve true si el Plugin contiene interacciones
Interfaces
Devuelve la lista de interfaces que este IPlugin implementa.
IsBeta
Devuelve true si el plugin es Beta
IsUnsafe
Determina si el plugin es inseguro
IsUnstable
Determina si el plugin es inseguro
License
Devuelve la licencia del IPlugin
MinMCARTVersion
Determina la versión mínima de MCART necesaria para este IPlugin
MyAssembly
Devuelve la referencia al ensamblado que contiene a este IPlugin.
Name
Devuelve el nombre del IPlugin
PluginInteractions
Devuelve una colección de opciones de interacción.
Reporter
Obtiene o establece el objeto ITaskReporter asociado a este IPlugin
Tag
Contiene información adicional de este plugin
TargetMCARTVersion
Determina la versión objetivo de MCART para este IPlugin
UIMenu
Convierte el PluginInteractions en un MenuItem.
Version
Devuelve la versión del IPlugin
