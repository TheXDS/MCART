TaskReporterPlugin (Eventos)

[Ésta es documentación preliminar y está sujeta a cambios.]

El tipo TaskReporterPlugin expone a los siguientes miembros.

Eventos
  NombreDescripción
Begun
Se genera cuando se ha iniciado una tarea.
CancelRequested
Se genera cuando este TaskReporter ha solicifado la detención de la tarea en ejecucion.
Ended
Se genera cuando una tarea ha finalizado.
Error
Se genera cuando ocurre una excepción durante la ejecución de la tarea.
PluginFinalized
Se produce cuando un Plugin ha sido finalizado.
(Se hereda de Plugin.)
PluginFinalizing
Se produce cuando un Plugin va a ser finalizado.
(Se hereda de Plugin.)
PluginLoaded
Se produce cuando un Plugin ha sido cargado.
(Se hereda de Plugin.)
PluginLoadFailed
Se produce cuando un Plugin no pudo ser cargado.
(Se hereda de Plugin.)
Reporting
Se genera cuando una tarea desea reportar su estado.
Stopped
Se genera cuando una tarea ha sido cancelada.
TaskTimeout
Se genera cuando se ha agotado el tiempo de espera establecido para ejecutar la tarea.
UIChangeRequested
Se produce cuando un Plugin solicita que se actualice su interfaz gráfica, en caso de contenerla.
(Se hereda de Plugin.)
