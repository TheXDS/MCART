Define una serie de métodos y propiedades a implementar por un control que permita mostrar gráficos de cualquier tipo.

Describe una serie de métodos a implementar por clases que permitan colorear una colección de IGraphData.
IGraphData
Determina una serie de propiedades a implementar por clases que puedan ser utilizadas en controles/widgets IGraph.
ISliceGraph
Expone una serie de métodos de redibujado disponibles para un control que acepte un Slice.
