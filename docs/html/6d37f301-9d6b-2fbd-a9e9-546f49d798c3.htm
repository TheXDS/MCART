IEasyCryptoPlugin (Métodos)

[Ésta es documentación preliminar y está sujeta a cambios.]

El tipo IEasyCryptoPlugin expone a los siguientes miembros.

Métodos
  NombreDescripción
Decrypt
Descifra el mensaje encriptado especificado.
(Se hereda de IEasyCrypto.)
Encrypt
Encripta el texto especificado.
(Se hereda de IEasyCrypto.)
Init
Inicializa la instancia del motor de encriptado utilizando la contraseña y el vector de inicialización especificados.
(Se hereda de IEasyCrypto.)
MinRTVersion
Determina la versión mínima de MCART necesaria para este IPlugin
(Se hereda de IPlugin.)
RequestUIChange
Solicita a la aplicación que se actualize la interfaz de interacción del IPlugin
(Se hereda de IPlugin.)
TargetRTVersion
Determina la versión objetivo de MCART necesaria para este IPlugin
(Se hereda de IPlugin.)
UIPanelT, PanelT
Convierte el PluginInteractions en un Panel.
(Se hereda de IPlugin.)
