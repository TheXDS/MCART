Point (Métodos)

[Ésta es documentación preliminar y está sujeta a cambios.]

El tipo Point expone a los siguientes miembros.

Métodos
  NombreDescripción
Angle
Calcula el ángulo formado por la línea que intersecta el orígen y este Point contra el eje horizontal X.
Equals
Indica si esta instancia y un objeto especificado son iguales.
(Reemplaza a ValueTypeEquals(Object).)
FitsInBox(Point, Point)
Determina si el punto se encuentra dentro del rectángulo formado por los puntos especificados.
FitsInBox(Double, Double, Double, Double)
Determina si el punto se encuentra dentro del rectángulo formado por las coordenadas especificadas.
GetHashCode
Devuelve el código Hash de esta instancia.
(Reemplaza a ValueTypeGetHashCode.)
GetType (Se hereda de Object.)
Magnitude
Calcula la magnitud de las coordenadas.
Magnitude(Point)
Calcula la magnitud de las coordenadas desde el punto especificado.
Magnitude(Double, Double)
Calcula la magnitud de las coordenadas desde el punto especificado.
