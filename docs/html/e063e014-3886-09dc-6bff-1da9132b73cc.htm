MCART.Events (Espacio de nombres)

[Ésta es documentación preliminar y está sujeta a cambios.]

El espacio de nombres MCART.Attributes contiene múltiples clases que pueden ser utilizadas como argumentos de eventos acompañados de sus respectivos delegados para diversos fines.
Clases
  ClaseDescripción
DependencyPropertyChangingEventArgs
Contiene información para los eventos basados en DependencyPropertyChangingEventHandler.
ExceptionEventArgs
Incluye información de evento para cualquier clase con eventos de excepción.
IncommingDataEventArgs
Incluye información de evento para cualquier clase con eventos de recepción de datos.
ItemSavedEventArgs
Contiene información de evento para cualquier clase con eventos donde se guardó información.
ItemSavingEventArgs
Contiene información de evento para cualquier clase con eventos donde se guardará información.
LoggingEventArgs
Incluye información de evento para cualquier clase con eventos de logging (bitácora).
ProgressionEventArgs
Incluye informaicón de evento para cualquier clase con eventos que reporten el progreso de una operación.
ValueChangingEventArgsT
Incluye informaicón para cualquier evento que incluya tipos de valor y puedan ser cancelados.
ValueEventArgsT
Incluye informaicón de evento para cualquier clase con eventos que incluyan tipos de valor.
Delegados
  DelegadoDescripción
DependencyPropertyChangingEventHandler
Delegado que controla un evento cancelable de cambio en el valor de una propiedad de dependencia.
