TheXDS.MCART.Resources (Espacio de nombres)

[Ésta es documentación preliminar y está sujeta a cambios.]

Este espacio de nombres contiene todos los recursos incrustados de MCART, como ser cadenas, gráficos, colores, etc.
Clases
  ClaseDescripción
AssemblyUnpackerT
Clase base que permite definir un IUnpackerT que extrae recursos incrustados desde un Assembly.
Colors
Incluye una colección de colores adicionales.
DeflateGetter
ICompressorGetter que construye un DeflateStream que puede ser utilizado para extraer información comprimida desde un Stream.
GZipGetter
ICompressorGetter que construye un GZipStream que puede ser utilizado para extraer información comprimida desde un Stream.
Icons
Contiene íconos y otras imágenes para utilizar en aplicaciones de Windows Presentation Framework.
ImageUnpacker
AssemblyUnpackerT que extrae recursos de imagen como un BitmapImage.
RTInfo
Contiene métodos con funciones de identificación en información del ensamblado de MCART.
Strings
Contiene cadenas de texto genéricas, además de funciones de composición de texto.
StringUnpacker
AssemblyUnpackerT que permite extraer archivos de texto incrustados en un ensamblado como una cadena.
Unpacker
AssemblyUnpackerT que expone directamente los Stream de los recursos incrustados de un ensamblado.
Interfaces
  InterfazDescripción
IAsyncUnpackerT
Define una serie de métodos a implementar por una clase que permita obtener y extraer recursos de forma asíncrona.
ICompressorGetter
Define una serie de métodos a implementar por una clase que permita obtener un Stream para extraer información comprimida desde otro Stream.
IUnpackerT
Define una serie de métodos a implementar por una clase que permita obtener y extraer recursos.
Enumeraciones
  EnumeraciónDescripción
IconsIconID
Determina el ícono a obtener.
