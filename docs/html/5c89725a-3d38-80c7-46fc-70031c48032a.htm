Parámetros

lst
Tipo: System.Collections.GenericIEnumerableInt32
Colección a procesar.
baseZero (Optional)
Tipo: SystemBoolean
Opcional. si es true, la base de porcentaje es cero; de lo contrario, se utilizará el valor mínimo dentro de la colección.

Valor devuelto

Tipo: IEnumerableSingle
Una colección de Single con sus valores expresados en porcentaje.

Nota sobre el uso

En Visual Basic y C#, se puede llamar a este método como a un método de instancia en cualquier objeto de tipo IEnumerableInt32. Cuando emplee la sintaxis de método de instancia para llamar a este método, omita el primer parámetro. Para obtener más información, vea Métodos de extensión (Visual Basic) o Métodos de extensión (Guía de programación C#).
