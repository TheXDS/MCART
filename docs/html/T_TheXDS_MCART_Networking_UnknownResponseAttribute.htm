Observaciones
Si ningún miembro de la enumeración se marca con este atributo, en caso de no existir el comando de la solicitud, se devolverá el valor que sea marcado con el atributo ErrorResponseAttribute, o en su defecto se lanzará una excepción que el servidor deberá manejar.
