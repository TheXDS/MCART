Parámetros

timeout
Tipo: SystemTimeSpan
Indica el tiempo total de espera antes de generar el evento TaskTimeout.
genTOutEx (Optional)
Tipo: SystemBoolean
Parámetro opcional. Si es true, se generará un TimeoutException al agotarse el tiempo de espera especificado durante una tarea. Si se omite, o se establece en false, no se generará la excepción. El evento TaskTimeout se genera indistintamente de este valor.

Implementa

ITaskReporterBegin(TimeSpan, Boolean)
Observaciones
Para que una tarea pueda detenerse, ésta debe monitorear el valor de CancelPending por cada vuelta del ciclo, y tomar las acciones necesarias para finalizar. Opcionalmente puede manejarse la detención de la tarea por medio del evento CancelRequested.
