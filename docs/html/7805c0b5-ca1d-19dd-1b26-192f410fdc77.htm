CommonCouldItBe (Método )

[Ésta es documentación preliminar y está sujeta a cambios.]

Comprueba si un nombre podría tratarse de otro indicado.

Espacio de nombres:  MCART
Ensamblado:  MCART (en MCART.dll) Versión: 0.7.0.0

public static double CouldItBe(
	this string checkName,
	string actualName,
	double tolerance = 0.75
)
<ExtensionAttribute>
Public Shared Function CouldItBe ( 
	checkName As String,
	actualName As String,
	Optional tolerance As Double = 0.75
) As Double

Parámetros

checkName
Tipo: SystemString
Nombre a comprobar.
actualName
Tipo: SystemString
Nombre real conocido.
tolerance (Optional)
Tipo: SystemDouble
Opcional. Double entre 0.0 y 1.0 que establece el nivel mínimo de similitud aceptado. si no se especifica, se asume 75% (0.75).

Valor devuelto

Tipo: Double
Un valor que representa la probabilidad de que checkName haga referencia al nombre actualName.

Nota sobre el uso

En Visual Basic y C#, se puede llamar a este método como a un método de instancia en cualquier objeto de tipo String. Cuando emplee la sintaxis de método de instancia para llamar a este método, omita el primer parámetro. Para obtener más información, vea Métodos de extensión (Visual Basic) o Métodos de extensión (Guía de programación C#).
Excepciones
ExcepciónCondición
ArgumentOutOfRangeException Se produce cuando tolerance no es un valor entre 0.0 y 1.0.
ArgumentNullException Se produce cuando checkName o actualName son cadenas vacías o null.
