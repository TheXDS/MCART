Colección de InteractionItem del Plugin.

Valor de propiedad

Tipo: ObservableCollectionInteractionItem
Observaciones
Esta colección contendrá todas las interacciones asociadas al Plugin, tanto las que MCART cablea automáticamente, como la que sean agregadas durante la ejecución.
Ejemplos
En este ejemplo se muestran distintos métodos para agregar elementos de interacción a un Plugin:

 2{
 3    // Este es un elemento de interacción que MCART cableará automáticamente.
 4    [Name("Interacción 1")]
 5    [Description("Muestra un mensaje de interacción.")]
 6    [InteractionItem] // <=- Este atributo marca a este método para cablearse automáticamente.
 7    public void Interaccion1(object sender, System.EventArgs e)
 8    {
 9        System.Diagnostics.Debug.Print("Interacción 1 ejecutada.");
10    }
11
12    // Este es un elemento de interacción cableado manualmente.
13    public void Interaccion2(object sender, System.EventArgs e)
14    {
15        System.Diagnostics.Debug.Print("Interacción 2 ejecutada.");
16    }
17
18    // Este es un elemento de interacción cableado manualmente, 
19    // con atributos que especifican el nombre y la descripción.
20    [Name("Interacción 3")]
21    [Description("Muestra un tercer mensaje de interacción.")]
22    public void Interaccion3(object sender, System.EventArgs e)
23    {
24        System.Diagnostics.Debug.Print("Interacción 3 ejecutada.");
25    }

27    /// <summary>
28    /// Constructor del Plugin.

30    public MyPlugin()
31    {
32        // la primera interacción es configurada y añadida de forma automática
33        // por MCART, por lo que no es necesario realizar ninguna acción.
34
35        // Crear una segunda interacción...
36        var Interac2 = new InteractionItem(Interaccion2,
37            "Interacción 2", 
38            "Muestra un segundo mensaje en la salida de la depuración.");
39
40        // Agregar la interacción al menú de este plugin...
41        uiMenu.Add(Interac2);
42
43
44        // Alternativamente, pueden establecerse atributos a la acción de

47
48
49        // También es posible utilizar delegados o lambdas con firma compatible
50        // con System.EventHandler
51        uiMenu.Add(

54                System.Diagnostics.Debug.Print("Interacción 4 ejecutada.");
55            },
56            "Interacción 4", "Muestra otro mensaje en la salida de la depuración."));
