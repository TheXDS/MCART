TheXDS.MCART.Networking.Server (Espacio de nombres)

[Ésta es documentación preliminar y está sujeta a cambios.]

Este espacio de nombres provee múltiples clases que permiten crear aplicaciones de red. Aquí se encuentran las clases de lado del servidor.
Clases
  ClaseDescripción
Client
Representa a un cliente que no requiere datos de estado que se ha conectado al servidor.
ClientT
Representa un cliente que requiere datos de estado asociados que se ha conectado al servidor.
SelfWiredCommandProtocolTClient, TCommand, TResponse
Clase base para crear protocolos simples con bytes de comandos.
Server
Controla conexiones entrantes y ejecuta protocolos sobre los clientes que se conecten al servidor.
ServerTClient
Controla conexiones entrantes y ejecuta protocolos sobre los clientes que se conecten al servidor.
ServerProtocol
Esta clase abstracta determina una serie de funciones a heredar por una clase que provea de protocolos a un servidor.
ServerProtocolT
Esta clase abstracta determina una serie de funciones a heredar por una clase que provea de protocolos a un servidor.
SimpleProtocol
Define un protocolo súmamente simple.

  InterfazDescripción

Determina una serie de funciones a implementar por una clase que provea de protocolos a un servidor.
