Observaciones
Este método se creó debido a un Quirk de funcionamiento del método CreateDelegate(Type, Object, String, Boolean, Boolean), en el cual el mismo aún podría arrojar una excepción cuando no es posible enlazar un método a un delegado si el método contiene parámetros genéricos.
