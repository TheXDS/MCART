[Ésta es documentación preliminar y está sujeta a cambios.]

Realiza una mezcla entre los colores especificados.

Espacio de nombres:  MCART.Types.Extensions
Ensamblado:  MCART (en MCART.dll) Versión: 0.7.0.0

<ExtensionAttribute>
Public Shared Function Blend ( 
	c1 As Color,
	c2 As Color
) As Color

Parámetros

c1
Tipo: MCART.TypesColor
El primer Color a mezclar.
c2
Tipo: MCART.TypesColor
El segundo Color a mezclar.

Valor devuelto

Tipo: Color
Una mezcla entre los colores c1 y c2.

Nota sobre el uso

En Visual Basic y C#, se puede llamar a este método como a un método de instancia en cualquier objeto de tipo Color. Cuando emplee la sintaxis de método de instancia para llamar a este método, omita el primer parámetro. Para obtener más información, vea Métodos de extensión (Visual Basic) o Métodos de extensión (Guía de programación C#).
