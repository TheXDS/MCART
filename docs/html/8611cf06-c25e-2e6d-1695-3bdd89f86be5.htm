UI (Métodos)

[Ésta es documentación preliminar y está sujeta a cambios.]

El tipo UI expone a los siguientes miembros.

Métodos
  NombreDescripción
AllocConsole
Abre una consola para la aplicación.
Clear
Limpia el texto del control.
ClearWarn
Quita el estado de error de un control.
CollapseControls(IEnumerableUIElement)
Establece la propiedad Visibility a Collapsed a una lista de controles.
CollapseControls(UIElement)
Establece la propiedad Visibility a Collapsed a una lista de controles.
DisableControls(IEnumerableUIElement)
Deshabilita una lista de controles.
DisableControls(UIElement)
Deshabilita una lista de controles.
EnableControls(IEnumerableUIElement)
Habilita una lista de controles.
EnableControls(UIElement)
Habilita una lista de controles.
FlushBitmapEncoders
Limpia la lista en caché de los BitmapEncoder cargados desde el AppDomain actual.
FreeConsole
Libera la consola de la aplicación.
GetBitmap
Obtiene una imagen a partir de un Stream.
GetBitmapEncoders
Devuelve una colección de los códecs de mapas de bits disponibles. Soporta cargar códecs desde cualquier ensamblado cargado.
GetCircleArc(Double, Double, Double)
Genera un arco de círculo que puede usarse en Windows Presentation Framework.
GetCircleArc(Double, Double, Double, Double)
Genera un arco de círculo que puede usarse en Windows Presentation Framework.
GetDeviceCaps
Obtiene la información física del contexto del dispositivo gráfico especificado.
GetDpi
Obtiene las resolución horizontal y vertical de la pantalla en DPI.
GetDpi(IntPtr)
Obtiene las resolución horizontal y vertical de la ventana en DPI.
GetScalingFactor
Obtiene el factor de escala de la interfaz gráfica.
GetScalingFactor(IntPtr)
Obtiene el factor de escala de la ventana especificada por el IntPtrHwnd.
GetXDpi
Obtiene la resolución horizontal de la pantalla en DPI.
GetXDpi(IntPtr)
Obtiene la resolución horizontal de la ventana en DPI.
GetYDpi
Obtiene la resolución vertical de la pantalla en DPI.
GetYDpi(IntPtr)
Obtiene la resolución vertical de la ventana en DPI.
HideControls(IEnumerableUIElement)
Establece la propiedad Visibility a Hidden a una lista de controles.
HideControls(UIElement)
Establece la propiedad Visibility a Hidden a una lista de controles.
IsWarned
Obtiene un valor que determina si el control está advertido.
PickBrush
Devuelve un Brush aleatorio.
Render(FrameworkElement)
Crea un mapa de bits de un FrameworkElement.
Render(FrameworkElement, Int32)
Crea un mapa de bits de un FrameworkElement.
Render(FrameworkElement, Size, Int32)
Crea un mapa de bits de un FrameworkElement.
Render(FrameworkElement, Size, Size, Int32)
Crea un mapa de bits de un FrameworkElement estableciendo el tamaño en el cual se dibujará el control, por lo que no necesita haberse mostrado en la interfaz de usuario.
ShowControls(IEnumerableUIElement)
Establece la propiedad Visibility a Visible a una lista de controles.
ShowControls(UIElement)
Establece la propiedad Visibility a Visible a una lista de controles.
ToggleControls(IEnumerableUIElement)
Habilita o deshabilita una lista de controles según su estado previo.
ToggleControls(UIElement)
Habilita o deshabilita una lista de controles según su estado previo.
ToImage
Convierte un BitmapSource en un BitmapImage.
Warn
Establece un estado de error para un control.
