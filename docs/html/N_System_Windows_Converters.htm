System.Windows.Converters (Espacio de nombres)

[Ésta es documentación preliminar y está sujeta a cambios.]

Este espacio de nombres contiene clases de conversión de valores utilizadas en propiedades de dependencia en Windows Presentation Framework.
Clases
  ClaseDescripción
AddConverter
Permite la adición de propiedades numéricas.
AllVisibileConverter
Determina un valor de visibilidad basado en que todos los valores sean igual a Visible.
AnyContentVisibilityConverter
Para un ContentControl o un Panel , obtiene un valor Visible si al menos un control hijo directo es visible.
AnyVisibileConverter
Determina un valor de visibilidad basado en que exista al menos un valor igual a Visible.
BooleanConverterT
Clase base para crear convertidores de valores booleanos.
BooleanInverter
Invierte un valor booleano
BooleanToBlurEffectConverter
Genera un efecto de desenfoque basado en una condición booleana.
BooleanToInvVisibilityConverter
Inverso de BooleanToVisibilityConverter
BoolFlagConverterT
Clase base para crear convertidores de valores booleanos que analizan banderas de una enumeración.
BrushOpacityAdjust
Permite compartir un recurso de Brush entre controles, ajustando la opacidad del enlace de datos.
CountVisibilityConverter
Convierte un valor Int32 a Visibility
DoubleMarginConverter
Convierte un Double en un Thickness.
DoubleVisibilityConverter
Convierte un valor Double a Visibility.
FloatConverterBase
Clase base que incluye un método para obtener un Single.
FloatVisibilityConverter
Convierte un valor Single a Visibility.
FromStringConverter
Convierte un valor a su representación como un String.
HealthBrushConverter
Obtiene un Brush correspondiente a la salud expresada com porcentaje.
HeatBrushConverter
Obtiene un Brush correspondiente a la salud expresada com porcentaje.
InverterT
Clase base para crear convertidores de valores que inviertan el valor de una propiedad de dependencia.
LabeledDoubleConverter
Convierte un Double en un String, opcionalmente mostrando una etiqueta si el valor es inferior a cero.
MaxSizeVisibilityConverter
Oculta un control si su tamaño es superior al umbral especificado.
MaxValueVisibilityConverter
Convierte un valor Double en un Visibility
MinSizeVisibilityConverter
Oculta un control si su tamaño es inferior al umbral especificado.
MinValueVisibilityConverter
Convierte un valor Double en un Visibility
MultiplyConverter
Permite la multiplicación de propiedades numéricas.
NotNullToBooleanConverter
Devuelve si el elemento a convertir no es
NotNullToVisibilityConverter
Devuelve Visible si el elemento a convertir no es
NullBoolConverterT
Clase base para crear convertidores de valores booleanos que pueden ser .
NullToBooleanConverter
Devuelve si el elemento a convertir es
NullToVisibilityConverter
Devuelve Visible si el elemento a convertir es
NumberToBooleanConverter
Convierte directamente un número a Boolean
SizeVisibilityConverter
Clase base para convertidores de valores que tomen un valor Size para determinar un valor de tipo Visibility basado en un umbral.
StrictTypeBooleanConverter
Determina si un objeto es una instancia del tipo provisto, y devuelve si lo es.
StrictTypeVisibilityConverter
Determina si un objeto es una instancia del tipo provisto, y devuelve Visible si lo es.
StringVisibilityConverter
Convierte un String a un Visibility.
ThresholdConverterTIn, TOut
Clase base para un convertidor de valores que digitalice un valor arbitrario de entrada.
ToStringConverter
Convierte un valor a su representación como un String.
TypeBooleanConverter
Determina si un objeto es una instancia del tipo provisto o de una clase derivada, y devuelve si lo es.
TypeVisibilityConverter
Determina si un objeto es una instancia del tipo provisto o de una clase derivada, y devuelve Visible si lo es.
VisibilityInverter
Invierte un valor de Visibility.
