IPluginChecker (Métodos)

[Ésta es documentación preliminar y está sujeta a cambios.]

El tipo IPluginChecker expone a los siguientes miembros.

Métodos
  NombreDescripción
HasT
Comprueba si un ensamblado contiene un IPlugin del tipo especificado.
IsCompatible
Comprueba que el tipo cargado sea compatible con esta versión de MCART.
IsVaild(Assembly)
Comprueba si un ensamblado contiene clases cargables como IPlugin.
IsVaild(Type)
Comprueba si un tipo es una clase cargable como IPlugin.
