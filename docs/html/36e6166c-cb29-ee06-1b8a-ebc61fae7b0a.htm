MCART.Exceptions (Espacio de nombres)

[Ésta es documentación preliminar y está sujeta a cambios.]

El espacio de nombres MCART.Attributes contiene múltiples clases que pueden ser utilizadas como excepciones para diversos fines.
Clases
  ClaseDescripción
ConnectionClosedException
Excepción que se produce cuando la conexión se encontraba cerrada al intentar enviar o recibir datos
CorruptDataException
Excepción que se produce cuando se reciben datos corruptos
CouldntConnectException
Excepción que se produce cuando no se puede realizar la conexión
DataAlreadyExistsException
Excepción que se produce al intentar crear nueva información dentro de una Database con un Uid que ya existe
DataNotFoundException
Excepción que se produce al no encontrar los datos solicitados.
DirectoryIsFullException
Excepción que se produce cuando se determina que el directorio está lleno
EmptyCollectionExceptionT
Excepción que se produce cuando un IEnumerableT está vacío.
FeatureNotAvailableException
Excepción que se produce al intentar utilizar una característica no disponible
InsufficientDataException
Excepción que se produce al no haber suficientes datos
InterfaceExpectedException
Excepcion que se produce cuando un método o función esperaba una interfaz como argumento.
InterfaceNotImplementedException
Excepcion que se produce cuando se solicita cargar un Plugin con la interfaz especificada, pero no la implementa.
InvalidDatabaseException
Excepción que se produce cuando una base de datos no es válida.
InvalidMethodSignatureException
Excepción que se produce cuando la firma de un método representado en un MethodInfo no es válida.
InvalidPasswordException
Excepción que se produce si la contraseña es incorrecta
InvalidPluginClassException
Excepción que se produce cuando la clase no puede ser cargada como Plugin
InvalidReturnValueException
Excepción que se produce cuando se detecta que una función ha devuelto un valor inválido sin generar una excepción.
InvalidTypeException
Excepción que se produce al hacer referencia a un tipo inválido
MissingArgumentException
Excepción que se produce cuando falta un argumento
NotMyDatabaseException
Excepción que se produce cuando una base de datos no corresponde a la aplicación.
NotPluginException
Excepción que se produce al intentar cargar plugins desde un ensamblado que no contiene ninguno.
OperationException
Excepción que se produce cuando una operación falla
PluginClassNotFoundException
Excepción que se produce cuando un Assembly no contiene la clase especificada.
PluginException
Excepción que produce un Plugin al encontrar un error
PluginInitializationException
Excepción que se produce cuando un Plugin encuentra un error al inicializarse.
PluginNeededException
Excepción que se produce cuando se requere un plugin para continuar
PluginNotInitializedException
Excepción que se produce cuando se llama a un método de un Plugin sin inicializar
ServerNotFoundException
Excepción que se produce cuando no se puede resolver un nombre DNS, o no se encuentra el servidor especificado.
StackUnderflowException
Excepción que se produce al intentar remover un objeto de una pila vacía
TooFewArgumentsException
Excepción que se produce al enviar muy pocos parámetros a un método
TooManyArgumentsException
Excepción que se produce al enviar demasiados parámetros a un método
UnknownTypeException
Excepción que se produce al hacer referencia a un tipo desconodico
