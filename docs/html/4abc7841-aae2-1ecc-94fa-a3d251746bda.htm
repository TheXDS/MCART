MCART.PluginSupport (Espacio de nombres)

[Ésta es documentación preliminar y está sujeta a cambios.]

Este espacio de nombres contiene la funcionalidad completa de administración y soporte de plugins por medio de la implementación de MCART.
Clases
  ClaseDescripción
InteractionItem
Esta clase define a un elemento de interacción.
InteractionItemAttribute
Marca un método como un elemento de interacción a ser cargado automáticamente.
Plugin
Clase base para todos los plugins que puedan ser contruídos y administrador por MCART.
PluginChecker
Esta clase realiza diferentes verificaciones de compatibilidad de plugins.
PluginFinalizedEventArgs
Incluye información adicional del evento PluginFinalized.
PluginFinalizingEventArgs
Incluye información adicional del evento PluginFinalizing.
PluginLoadedEventArgs
Incluye información adicional del evento PluginLoaded.
PluginLoader
Permite cargar IPlugin.
RelaxedPluginChecker
PluginChecker con reglas de compatibilidad relajadas.
StrictPluginChecker
PluginChecker con reglas de compatibilidad estrictas.
UIChangeEventArgs
Incluye información adicional del evento UIChangeRequested
Interfaces
  InterfazDescripción
IPlugin
Define una interfaz básica para crear Plugins administrados por MCART
IPluginChecker
Define una serie de métodos y propiedades a implementar por una clase que pueda utilizarse para verificar la validez de los IPlugin que se intenten cargar.
IPluginLoader
Define una serie de métodos y propiedades a implementar por una clase que pueda utilizarse para cargar clases que implementen la interfaz IPlugin.
Delegados
  DelegadoDescripción
PluginFinalizedEventHandler

PluginFinalizingEventHandler
Maneja el evento PluginFinalizing.
PluginLoadedEventHandler
Maneja el evento PluginLoaded.
PluginLoadFailedEventHandler
Maneja el evento PluginLoadFailed.
UIChangeRequestedEventHandler
Maneja el evento UIChangeRequested.
Enumeraciones
  EnumeraciónDescripción
FinalizingReason
Determina el motivo de finalización del Plugin.
