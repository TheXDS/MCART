Parámetros de tipo

Tipo de controles que serán generados por cada InteractionItem que el Plugin contiene.
PanelT
Tipo de panel que contendrá las interacciones.

Valor del campo

Tipo: PanelT
Un Panel que puede agregarse a un Menu de Windows Presentation Framework.

Valor devuelto

Tipo: PanelT
Un PanelT que contiene las interacciones contenidas por el IPlugin, en la propiedad PluginInteractions.
