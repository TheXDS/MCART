Escribe un conjunto de objetos Byte en el Stream.

Espacio de nombres:  MCART.Types.Extensions
Ensamblado:  MCART (en MCART.dll) Versión: 0.7.0.0

Parámetros

fs
Tipo: System.IOStream
Stream del cual este método es una extensión.
bytes
Tipo: SystemByte
Colección de objetos Byte a escribir en el Stream.

Nota sobre el uso

En Visual Basic y C#, se puede llamar a este método como a un método de instancia en cualquier objeto de tipo Stream. Cuando emplee la sintaxis de método de instancia para llamar a este método, omita el primer parámetro. Para obtener más información, vea Métodos de extensión (Visual Basic) o Métodos de extensión (Guía de programación C#).
