MCART.Types.TaskReporter (Espacio de nombres)

[Ésta es documentación preliminar y está sujeta a cambios.]

Este espacio de nombres es temporal, y contiene el tipo TaskReporter y otras clases relacionadas.
Clases
  ClaseDescripción
BegunEventArgs
Contiene información del evento Begun.
ConsoleTaskReporter
Implementación de TaskReporter que muestra el progreso de una tarea en la consola del sistema.
DummyTaskReporter
TaskReporter que no implementa ningún medio de interacción con el usuario.
ProgressEventArgs
Contiene información del evento Reporting.
ProgressEventArgsT
Contiene información del evento Reporting.
TaskReporter
Clase base para objetos que puedan utilizarse para reportar el progreso de una operación o tarea.
TaskReporterControl
Clase base para los controles de Windows Presentation Framework que pueden utilizarse para mostrar el progreso de una tarea por medio de la interfaz ITaskReporter.
TaskReporterPlugin
Implementa la interfaz ITaskReporter para crear plugins compatibles con MCART que permitan reportar el progreso de una operación o tarea.
Interfaces
  InterfazDescripción
ITaskReporter
Define una serie de métodos a implementar por una clase que permita a una tarea reportar el progreso de una operación, generalmente cíclica.
Delegados
  DelegadoDescripción
BegunEventHandler
Se produce cuando una tarea se ha iniciado
CancelRequestedEventHandler
Indica a la tarea que se ha solicitado que se detenga
EndedEventHandler
Se produce cuando una tarea finalizó correctamente
ErrorEventHandler
Se produce cuando una tarea indica que finalizó con error
ForAction
Define un delegado que ejecutará una acción cíclica con la estructura de un ciclo for.
ForEachActionT
Define un delegado que ejecutará una acción cíclica con la estructura de un ciclo foreach.
ReportingEventHandler
Se produce cuando la tarea desea reportar su estado
StoppedEventHandler
Se produce cuando una tarea es cancelada
TaskTimeoutEventHandler
Se produce cuando se ha agotado el tiempo de espera para una tarea
