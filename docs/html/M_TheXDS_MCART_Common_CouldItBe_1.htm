Parámetros

checkName
Tipo: SystemString
Nombre a comprobar.
actualName
Tipo: SystemString
Nombre real conocido.
tolerance
Tipo: SystemSingle
Opcional. Single entre 0.0 y 1.0 que establece el nivel mínimo de similitud aceptado. si no se especifica, se asume 75% (0.75).

Valor devuelto

Tipo: Single
Un valor que representa la probabilidad de que checkName haga referencia al nombre actualName.

Nota sobre el uso

En Visual Basic y C#, se puede llamar a este método como a un método de instancia en cualquier objeto de tipo String. Cuando emplee la sintaxis de método de instancia para llamar a este método, omita el primer parámetro. Para obtener más información, vea Métodos de extensión (Visual Basic) o Métodos de extensión (Guía de programación C#).
