MCART.Networking.Server (Espacio de nombres)

[Ésta es documentación preliminar y está sujeta a cambios.]

Este espacio de nombres contiene herramientas específicas para crear aplicaciones servidor que provean servicios de red.
Clases
  ClaseDescripción
Client
Representa a un cliente que no requiere datos de estado que se ha conectado al servidor.
ClientT
Representa un cliente que requiere datos de estado asociados que se ha conectado al servidor.
Protocol
Esta clase abstracta determina una serie de funciones a heredar por una clase que provea de protocolos a un servidor.
ProtocolTClient
Esta clase abstracta determina una serie de funciones a heredar por una clase que provea de protocolos a un servidor.
Server
Controla conexiones entrantes y ejecuta protocolos sobre los clientes que se conecten al servidor.
ServerTClient
Controla conexiones entrantes y ejecuta protocolos sobre los clientes que se conecten al servidor.
