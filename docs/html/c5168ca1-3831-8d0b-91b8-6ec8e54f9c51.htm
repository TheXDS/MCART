MCART.Types (Espacio de nombres)

[Ésta es documentación preliminar y está sujeta a cambios.]

El espacio de nombres MCART.Types contiene nuevos tipos con funcionalidad adicional y/o universal.
Estructuras
  EstructuraDescripción
Color
Estructura universal que describe un color en sus componentes alfa, rojo, verde y azul.
Point
Tipo universal para un conjunto de coordenadas bidimensionales.
Point3D
Tipo universal para un conjunto de coordenadas tridimensionales.
