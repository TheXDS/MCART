SelfWiredCommandClientTCommand, TResponse (Eventos)

[Ésta es documentación preliminar y está sujeta a cambios.]

El tipo genérico SelfWiredCommandClientTCommand, TResponse expone a los siguientes miembros.

Eventos
  NombreDescripción
Connected
Se produce cuando ocurre una conexión de manera satisfactoria.
(Se hereda de ClientBase.)
ConnectionFailed
Se produce si la conexión con el servidor ha fallado.
(Se hereda de ClientBase.)
ConnectionLost
Se produce cuando se pierde la conexión con el servidor inesperadamente.
(Se hereda de ActiveClient.)
ServerError
Se produce cuando el servidor envía un mensaje indicando estado de error.
UnknownCommandIssued
Se produce cuando el servidor envía un mensaje indicando que no reconoce al comando que se le ha enviado.
