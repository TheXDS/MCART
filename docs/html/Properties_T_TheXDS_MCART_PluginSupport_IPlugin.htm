IPlugin (Propiedades)

[Ésta es documentación preliminar y está sujeta a cambios.]

El tipo IPlugin expone a los siguientes miembros.

Propiedades
  NombreDescripción
Author
Devuelve el autor del IExposeInfo
(Se hereda de IExposeInfo.)
ClsCompliant
Obtiene un valor que indica si este IExposeInfo cumple con el Common Language Standard (CLS)
(Se hereda de IExposeInfo.)
Copyright
Devuelve el Copyright del IExposeInfo
(Se hereda de IExposeInfo.)
Description
Obtiene la descripción del elemento.
(Se hereda de IDescriptible.)
HasInteractions
Devuelve si el Plugin contiene interacciones
HasLicense
Obtiene un valor que determina si este IExposeInfo contiene información de licencia.
(Se hereda de IExposeInfo.)
Icon
Obtiene un ícono opcional a mostrar que describe al elemento.
(Se hereda de IExposeInfo.)
Interfaces
Devuelve la lista de interfaces que este IPlugin implementa.
IsBeta
Devuelve si el plugin es Beta
IsUnmanaged
Determina si el plugin es inseguro
IsUnstable
Determina si el plugin es inseguro
License
Devuelve la licencia del IExposeInfo
(Se hereda de IExposeInfo.)
MinMCARTVersion
Determina la versión mínima de MCART necesaria para este IPlugin
MyAssembly
Devuelve la referencia al ensamblado que contiene a este IPlugin.
Name
Obtiene el nombre del elemento.
(Se hereda de INameable.)
PluginInteractions
Devuelve una colección de opciones de interacción.
Tag
Contiene información adicional de este plugin
TargetMCARTVersion
Determina la versión objetivo de MCART para este IPlugin
UIMenu
Convierte el PluginInteractions en un MenuItem.
Version
Devuelve la versión del IExposeInfo
(Se hereda de IExposeInfo.)
