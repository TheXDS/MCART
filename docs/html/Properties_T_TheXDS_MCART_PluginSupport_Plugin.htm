Plugin (Propiedades)

[Ésta es documentación preliminar y está sujeta a cambios.]

El tipo Plugin expone a los siguientes miembros.

Propiedades
  NombreDescripción
Author
Obtiene el autor de este Plugin.
Copyright
Obtiene la cadena de Copyright de este Plugin.
Description
Obtiene la descripción de este Plugin.
HasInteractions
Indica si este Plugin contiene o no interacciones.
InteractionItems
Colección de InteractionItem del Plugin.
Interfaces
Obtiene una colección de las interfaces implementadas por este Plugin.
IsBeta
Determina si este Plugin es una versión Beta.
IsUnmanaged
Determina si este Plugin contiene código no administrado.
IsUnstable
Determina si este Plugin es considerado como inestable.
License
Obtiene el texto de la licencia de este Plugin.
MinMCARTVersion
Determina la versión mínima de MCART necesaria para este Plugin.
MyAssembly
Obtiene la referencia al emsamblado que contiene a este Plugin.
Name
Obtiene el nombre de este Plugin.
PluginInteractions
Contiene una lista de interacciones que este Plugin. provee para incluir en una interfaz gráfica.
Tag
Contiene un objeto de libre uso para almacenamiento de cualquier inatancia que el usuario desee asociar a este Plugin.
TargetMCARTVersion
Determina la versión objetivo de MCART para este Plugin.
UIMenu
Convierte el PluginInteractions en un MenuItem.
Version
Obtiene la versión de este Plugin.
