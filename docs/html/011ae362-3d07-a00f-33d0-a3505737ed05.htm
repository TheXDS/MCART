MCART.Forms (Espacio de nombres)

[Ésta es documentación preliminar y está sujeta a cambios.]

El espacio de nombres MCART.Controls contiene una librería de ventanas y diálogos disponibles para aplicaciones de GUI.
Clases
  ClaseDescripción
PasswordDialog
Esta clase contiene una ventana de Windows Presentation Framework que permite al usuario validar contraseñas, así como también establecerlas y medir el nivel de seguridad de la contraseña escogida.
PluginBrowser
Diálogo que permite mostrar información acerca de los Plugin cargables por MCART.
Delegados
  DelegadoDescripción
PasswordDialogLoginValidator
Delegado que permite verificar una contraseña sin cerrar el PasswordDialog.
