MCART.Attributes (Espacio de nombres)

[Ésta es documentación preliminar y está sujeta a cambios.]

El espacio de nombres MCART.Attributes contiene múltiples clases que pueden ser utilizadas como atributos para diversos fines.
Clases
  ClaseDescripción
AuthorAttribute
Establece el autor del elemento.
BetaAttribute
Marca un elemento como versión Beta.
BoolAttributeBase
Clase base para los atributos basados en valores booleanos.
CopyrightAttribute
Establece el autor del elemento.
DescriptionAttribute
Establece una descripción para el elemento.
EmbeededLicenseAttribute
Establece un archivo embebido de licencia a asociar con el elemento.
FloatAttributeBase
Clase base para los atributos basados en números.
IntAttributeBase
Clase base para los atributos basados en números.
LenghtyAttribute
Indica que un elemento podría tardar en ejecutarse, y por lo tanto, es capaz de reportar su progreso.
LicenseFileAttribute
Establece un archivo de licencia a asociar con el elemento.
LicenseTextAttribute
Establece el texto de licencia a asociar con el elemento.
MinMCARTVersionAttribute
Especifica la versión mínima de MCART requerida por el elemento.
NameAttribute
Establece un nombre personalizado para mostrar del elemento.
NotPluginAttribute
Marca una clase para no ser cargada como IPlugin, a pesar de implementar IPlugin.
ServerAttribute
Attributo que define a un servidor.
StubAttribute
Indica que un elemento no está completo, y que no posee la funcionalidad necesaria.
TargetMCARTVersionAttribute
Especifica la versión de MCART recomendada para el elemento.
TextAttributeBase
Clase base para los atributos basados en texto.
ThunkAttribute
Marca un elemento como método de Thunking (facilita la llamada de otros métodos).
UnsecureAttribute
Marca un elemento como inseguro.
UnstableAttribute
Marca un elemento como inestable.
UnusableAttribute
Marca un elemento como no utilizable.
VersionAttribute
Especifica la versión de un elemento
VersionAttributeBase
Clase base para los atributos basados en valores booleanos.
