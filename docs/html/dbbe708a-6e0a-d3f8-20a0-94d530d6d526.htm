MCART.Security.Password (Espacio de nombres)

[Ésta es documentación preliminar y está sujeta a cambios.]

Este espacio de nombres contiene herramientas de seguridad específicas para trabajar con contraseñas.
Clases
  ClaseDescripción
DefaultEnableAttribute
Indica si un elemento se acitva o no de forma predeterminada.
ExtraPointsAttribute
Indica que un elemento representará un puntaje extra en la evaluación.
Generators
Contiene funciones que generan contraseñas.
PonderationAttribute
Describe un nivel de ponderación.
PwEvalRule
Define una regla de evaluación de contraseñas. Esta clase no puede heredarse.
PwEvaluator
Define un evaluador de contraseñas. Esta clase no puede heredarse.
RuleSets
Contiene un conjunto de reglas de evaluación de contraseñas.
Estructuras
  EstructuraDescripción
PwDialogResult
Representa el resultado de un cuadro de diálogo PasswordDialog.
PwEvalResult
Contiene información sobre el resultado de la evaluación de una contraseña.
Interfaces
  InterfazDescripción
IPwEvaluator
Describe los métodos a implementar para crear clases que evalúen la seguridad de una contraseña.
Delegados
  DelegadoDescripción
PwEvalFunc
Delegado que define una función que evalúa contraseñas.
PwGenerator
Delegado que define una función que genera contraseñas.
Enumeraciones
  EnumeraciónDescripción
PonderationLevel
Determina el nivel de ponderación a aplicar a un objeto PwEvalRule
PwMode
Configura el cuadro de contraseña al utilizar la función ChoosePassword(PwMode, PwEvaluator, Int32).
