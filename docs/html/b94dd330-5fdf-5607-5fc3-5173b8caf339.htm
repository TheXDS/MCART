ObjectsHasAttrT (Método (MemberInfo, T))

[Ésta es documentación preliminar y está sujeta a cambios.]

Determina si un miembro posee un atributo definido.

Espacio de nombres:  MCART
Ensamblado:  MCART (en MCART.dll) Versión: 0.7.0.0
Sintaxis
C#
VB
Copy
public static bool HasAttr<T>(
	this MemberInfo member,
	out T attribute
)
where T : Attribute

<ExtensionAttribute>
Public Shared Function HasAttr(Of T As Attribute) ( 
	member As MemberInfo,
	<OutAttribute> ByRef attribute As T
) As Boolean

Parámetros

member
Tipo: System.ReflectionMemberInfo
Miembro del cual se extraerá el atributo.
attribute
Tipo: T
Parámetro de salida. Si un atributo de tipo T ha sido encontrado, el mismo es devuelto. Se devolverá null si el miembro no posee el atributo especificado.

Parámetros de tipo

T
Tipo de atributo a devolver. Debe heredar Attribute.

Valor devuelto

Tipo: Boolean
true si el miembro posee el atributo, false en caso contrario.

Nota sobre el uso

En Visual Basic y C#, se puede llamar a este método como a un método de instancia en cualquier objeto de tipo MemberInfo. Cuando emplee la sintaxis de método de instancia para llamar a este método, omita el primer parámetro. Para obtener más información, vea Métodos de extensión (Visual Basic) o Métodos de extensión (Guía de programación C#).
