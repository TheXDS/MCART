Clases
  ClaseDescripción
AuthorAttribute
Establece el autor del elemento.
BetaAttribute
Marca un elemento como versión Beta.
BoolAttribute
Clase base para los atributos basados en valores booleanos.
CompressorAttribute
Atributo que indica el compresor utilizado para este elemento.
CopyrightAttribute
Establece la información de Copyright del elemento.
DangerousAttribute
Indica que el uso de un elemento es peligroso.
DescriptionAttribute
Establece una descripción larga para este elemento.
EmbeededLicenseAttribute
Establece un archivo incrustado de licencia a asociar con el elemento.
FloatAttribute
Clase base para los atributos basados en números flotantes.
IdentifierAttribute
Indica una cadena que puede utilizarse para identificar a este elemento.
IntAttribute
Clase base para los atributos basados en números enteros.
LenghtyAttribute
Indica que un elemento podría tardar en ejecutarse.
LicenseFileAttribute
Establece un archivo de licencia externo a asociar con el elemento.
LicenseTextAttribute
Establece el texto de licencia a asociar con el elemento.
MaximumAttribute
Establece un valor máximo al cual se deben limitar los campos y propiedades.
MinimumAttribute
Establece un valor mínimo al cual se deben limitar los campos y propiedades.
MinMCARTVersionAttribute
Especifica la versión mínima de MCART requerida por el elemento.
NameAttribute
Establece un nombre personalizado para describir este elemento.
NotPluginAttribute
Marca una clase para no ser cargada como IPlugin, a pesar de implementar IPlugin.
ObjectAttribute
Clase base para los atributos de cualquier tipo.
ProtocolFormatAttribute
Establece un formato de protocolo para abrir un vínculo por medio del sistema operativo.
ServerAttribute
Attributo que define la ruta de un servidor.
StubAttribute
Indica un elemento cuyo propósito es simplemente el de reservar espacio para una posible expansión, y por lo tanto, actualmente no tiene ninguna funcionalidad.
TargetMCARTVersionAttribute
Especifica la versión de MCART recomendada para el elemento.
TextAttribute
Agrega un elemento textual genérico a un elemento, además de ser la clase base para los atributos que describan un valor representable como String para un elemento.
ThunkAttribute
Indica que un elemento es un proveedor de Thunking (facilita la llamada a otros elementos o miembros).
TypeAttribute
Agrega un elemento de tipo a un elemento, además de ser la clase base para los atributos que describan un valor representable como Type para un elemento.
UnmanagedAttribute
Indica que un elemento contiene código no administrado.
UnstableAttribute
Indica que un elemento contiene código que podría ser inestable.
UnusableAttribute
Marca un elemento como no utilizable.
VersionAttribute
Especifica la versión del elemento.
VersionAttributeBase
Especifica la versión de un elemento, además de ser la clase base para los atributos que describan un valor Version para un elemento.
Interfaces
  InterfazDescripción
IValueAttributeT
Define una interfaz para los atributos que expongan valores por medio de la propiedad Value.
